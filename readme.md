# Secure Password :

Overview

SecurePass is a simple, browser-based password manager and password generator built for personal use. It helps users generate strong passwords, check their strength, store credentials locally, and quickly access recently generated passwords â€” all without requiring accounts, signups, or internet access.

This project focuses on practicality, speed, and ease of use.


Features :

Generate strong, random passwords instantly
Customize password length and character types
Visual password strength indicator
Copy passwords to clipboard with one click
Store website/app credentials locally
View, copy, and delete stored passwords
Recently generated password history
Password security guidelines built in



Usage : 

Open the application
Generate a password using your preferred settings
Copy the password or store it with related details
Manage stored passwords from the dashboard
Reuse or delete passwords anytime
No login. No setup. Just open and use.
